Privacybeleid Mnemosyne

Laatst bijgewerkt: januari 2026

Mnemosyne is een app voor het bijhouden van je stemming en menstruatiecyclus. We nemen je privacy zeer serieus. Dit beleid legt uit welke gegevens we verzamelen en hoe we deze gebruiken.

1. Welke gegevens verzamelen we?

Mnemosyne verzamelt de volgende gegevens die je zelf invoert:

  • Stemmingsscores - Je dagelijkse stemmingswaarden (-5 tot +5)
  • Context tags - Tags zoals 'hoofdpijn', 'goed geslapen', 'stress'
  • Notities - Optionele persoonlijke notities bij entries
  • Menstruatiecyclus data - Cycluslengte, stopweek configuratie, menstruatie flow
  • Tijdstempels - Wanneer je entries hebt gemaakt
Belangrijk: Dit zijn bijzondere persoonsgegevens (gezondheidsdata) onder de Algemene Verordening Gegevensbescherming (AVG/GDPR).

2. Waarom verzamelen we deze gegevens?

We verzamelen deze gegevens uitsluitend om:

  • Je stemming en patronen te visualiseren
  • Correlaties te ontdekken tussen stemming en andere factoren
  • Je te helpen inzicht te krijgen in je welzijn
  • Je menstruatiecyclus te volgen en patronen te herkennen

We gebruiken je data NIET voor:

  • Advertenties of marketing
  • Verkoop aan derden
  • Analytics of tracking
  • Enige andere doeleinden dan hierboven beschreven

3. Waar wordt je data opgeslagen?

Je kunt kiezen waar je data wordt opgeslagen:

Optie 1: Alleen lokaal (aanbevolen voor maximale privacy)

  • Data blijft alleen op jouw apparaat
  • Niemand anders heeft toegang, ook wij niet
  • Data wordt beschermd door iOS apparaat-encryptie
  • Als je de app verwijdert, is de data weg

Optie 2: iCloud synchronisatie

  • Data wordt gesynchroniseerd via Apple iCloud naar je andere apparaten
  • Apple beheert de iCloud infrastructuur
  • Data blijft behouden als je de app herinstalleert
  • Apple kan technisch gezien toegang hebben tot deze data

Optie 3: Apple Health (optioneel)

  • Als je dit inschakelt, wordt stemmingsdata ook naar Apple Health geschreven
  • Dit is volledig optioneel
  • Apple Health data valt onder Apple's privacybeleid

4. Jouw rechten (AVG/GDPR)

Je hebt het recht om:

  • Inzage - Je data bekijken (via Instellingen → Privacy → Bekijk mijn data)
  • Dataportabiliteit - Je data exporteren als JSON bestand (via Instellingen → Privacy → Exporteer)
  • Verwijdering - Al je data verwijderen (via Instellingen → Privacy → Verwijder)
  • Intrekking toestemming - Je toestemming intrekken, waarna alle data wordt verwijderd

Al deze opties zijn direct beschikbaar in de app onder Instellingen → Privacy.

5. Beveiliging

  • Lokale data wordt beschermd door iOS apparaat-encryptie
  • We delen je data niet met derden
  • Er is geen analytics of tracking in de app
  • De app heeft geen eigen server - alle data is van jou
  • We verzamelen geen persoonlijk identificeerbare informatie

6. Kinderen

Mnemosyne is niet bedoeld voor kinderen onder de 16 jaar. We verzamelen niet bewust gegevens van kinderen.

7. Wijzigingen in dit beleid

Als we dit privacybeleid wijzigen, zullen we de "laatst bijgewerkt" datum bovenaan aanpassen. Bij significante wijzigingen zullen we je informeren via de app.
